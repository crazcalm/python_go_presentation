Section 1: It can be done!

We are going to write a go program. Make a shared object out of that program, and then call that program from python.

Shared Object file?

Shared libraries are libraries that are loaded by programs when they start. In terms of convention, these files are usually kept in a special place and given special names. You can read more about that at http://tldp.org/HOWTO/Program-Library-HOWTO/shared-libraries.html

For our purposes, we need to know that we can build shared object files via Go and the the name of these files end in ".so".

Lets build a hello world shared object file!

go build -o output_file.so -buildmode=c-shared source_file.go

Ctypes Module

Ctypes is a foreign function library for Python. It provides C compatible data types, and allows calling functions in DLLs or shared libraries. It can be used to wrap these libraries in pure Python.
